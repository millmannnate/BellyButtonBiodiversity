This project is researching the bacteria appearing in a human's belly button. It is led to believe that the bacteria from a human belly button can provide taste to a synthetic meat. The data given is a collection of samples from anonymous test subjects' belly buttons. Below we have plotted the data for the selected test subject number along with their demographic information.
